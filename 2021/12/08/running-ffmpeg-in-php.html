Running FFmpeg in a PHP script

08 Dec 2021 in Programming Notes Tags:

I spent a wasteful amount of time trying to use shell_exec() and retrieve the exit code, until I realized there was exec()

shell_exec() will return the output as a string, where as with exec(), you can get both the output and the exit code.

Here’s the code:

$output = [];
$result = 1;

exec('ffmpeg -i in.mp4 -hls_segment_filename \'segment%03d.ts\' out.m3u8', $output, $result)  

if($result === 0) {
    // Success: ...
} else {
    // Failure: report error and cleanup...
}
